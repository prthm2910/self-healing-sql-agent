<task type="auto">
  <name>Implement Decomposer and Worker Nodes</name>
  <files>src/workflow/nodes.py</files>
  <action>
    - Implement `decomposer_node(state: State, config: RunnableConfig)`:
      - Use the 8B Flash model (`small_model`) per D-DAC-01.
      - Use `with_structured_output(DecomposerOutput)`.
      - Prompt per the research blueprint: Question -> Sub-tasks + Join Plan.
      - Update state: `sub_tasks`, `join_plan`, and append thought/action to `agent_logs`.
    - Implement `worker_node(state: State, config: RunnableConfig)`:
      - Get current task from `state['sub_tasks'][state['current_task_index']]`.
      - Use `chat_model` (or `small_model`) to generate SQL for the atomic task.
      - Ensure `required_columns` are in the SELECT list.
      - Validate SQL with `sqlglot.parse_one(sql, read="postgres")`.
      - If validation fails, set `sql_error` for the `healer_node`.
      - Store snippet in `state['sql_snippets'][task_id]` and log to `agent_logs`.
  </action>
  <verify>
    <automated>python -c "from src.workflow.nodes import decomposer_node, worker_node; print('Nodes implemented')"</automated>
  </verify>
  <done>Manager and Worker nodes are implemented with structured logging and validation.</done>
</task>

  <action>
